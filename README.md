# Local Coin Settlement

# Getting Started

## Requirements

- [Nodejs](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)

## Quickstart

Install packages:

```
npm install
```

Compile:

```
npm run compile
```

Run Tests:

```
npm run test
```
